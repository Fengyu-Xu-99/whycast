# Curiosity Tree: Bloodhound Test

## Seed

为什么寻血猎犬长得这么特别，而且这么会追踪气味？

## Root Question

一只主要靠气味工作的狗，为什么会长成 Bloodhound 这样？

## Branches

### 1. Body Shape

- 为什么 Bloodhound 的脸这么长？
- 大鼻孔和长鼻腔对气味采样有什么帮助？
- 长耳朵、松皮、皱褶是不是和贴地追踪有关？
- 为什么 Bloodhound 看起来更重、更稳，而不是轻巧高速？

### 2. Smell And Tracking

- 狗是怎么从气味里读出路线的？
- 血、脚印、皮肤碎屑、体味、植物扰动分别会留下什么线索？
- 气味会怎么被风、湿度、温度、雨水、时间改变？
- 狗追踪时为什么会停、绕、回头、扩大搜索？

### 3. Bloodhound Skill

- Bloodhound 是不是狗里面“鼻子最灵”的？
- “最会闻”和“最会追踪”是不是同一件事？
- 它的强项是低浓度气味、个体气味，还是长距离路线追踪？
- 性格里的执着和工作能力有什么关系？

### 4. Scent Hounds

- 什么是 scent hound？
- Beagle、Basset Hound、Coonhound 和 Bloodhound 有哪些共同点？
- 为什么很多 scent hound 都有长耳朵、低头、耐力和叫声？
- 不同 scent hound 面对的猎物和地形有什么区别？

### 5. Sight Hounds

- sight hound 和 scent hound 的核心差别是什么？
- Greyhound、Whippet、Saluki、Borzoi 为什么更像速度机器？
- 长腿、深胸、收腰、轻窄身体分别服务什么任务？
- 靠视觉追猎和靠气味追踪，会塑造出怎样不同的身体？

### 6. Bigger WhyCast Question

- 狗不是泛泛地“会打猎”，而是不同任务的不同答案。
- 有的狗靠鼻子读过去，有的狗靠眼睛追现在。
- 一种动物长期被人类用来完成某种工作后，身体、感官、性格会怎么一起被塑造？

## Current Path

The selected path follows:

Bloodhound body → dog smell world → Bloodhound tracking skill → scent hounds → sight hounds → what dog bodies reveal about work.
